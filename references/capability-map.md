# Capability Map

This public skill is a fused system. It absorbs the strongest parts of several skill families without inheriting their private assumptions.

## Absorbed strengths

| Source family | What was absorbed | What was intentionally removed |
|---|---|---|
| Khazix-style writing | stronger openings, anti-report tone, selective rewrite, editorial smell test | fixed personal worldview, fixed creator identity |
| Baoyu skill set | image planning, illustration logic, infographic routing, publish closure mindset | any one locked provider or account |
| Huashu-style review | multi-pass proofreading, checklists, quality-gate thinking | private workflow wording and fixed internal context |
| WeChat skill stack | topic -> outline -> draft -> title -> score pipeline, DNA-first writing | private author presets and hardcoded brand assumptions |

## Final shape

What this skill tries to standardize:

1. planning
2. style calibration
3. drafting
4. review
5. repair
6. visual planning
7. publishing handoff

## Design philosophy

- modular, not monolithic
- portable, not private
- editorially opinionated, not author-locked
- handoff-driven, not prompt-spaghetti
