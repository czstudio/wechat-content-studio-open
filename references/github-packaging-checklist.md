# GitHub Packaging Checklist

Before publishing this skill to GitHub, verify:

## Remove private assumptions

- no private author names
- no private community names
- no private URLs or course names
- no fixed corpus paths
- no fixed publish account assumptions

## Keep what matters

- workflow
- handoff artifacts
- quality gates
- DNA merge rule
- portability guidance

## Make extension points explicit

- how to plug in image generation
- how to plug in publishing
- how to provide sample articles
- how to provide a style seed
- how to point to a local corpus

## README must explain

- who this is for
- what problem it solves
- why it is different from a simple writing prompt
- what the minimum inputs are
- what is optional
